## iTerm2 Color Schema

## Release: v0.0.1

## Getting started

1. Clone the repo or download the file `iterm2_theme.itermcolors`
2. *iTerm2 > Preferences > Profiles > Colors Tab*
3. Click *Color Presets...*
4. Click *Import...*
5. Select the `iterm2_theme` file
5. Select the *iterm2_theme* from *Load Presets...*
